```sh
npm install --save style.init.css
```

## 项目描述

* 项目使用 `postcss` 插件实现样式兼容，可供日常项目开发使用。
* 部分样式参考 [`normalize.css`](https://github.com/necolas/normalize.css)
* 有好的提议可以在 [`Issues`](https://github.com/kiccer/style.init.css/issues) 反馈给我，我会很乐意采纳。
