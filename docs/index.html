Style.init.css Demo

NPM

npm install --save style.init.css
# OR
yarn add style.init.css

Use

import 'style.init.css'

项目描述

  • 项目使用 postcss 插件实现样式兼容，可供日常项目开发使用。
  • 部分样式参考 normalize.css
  • 有好的提议可以在 Issues 反馈给我，我会很乐意采纳。

Project description百度翻译

  • The project uses postcss plug-in to achieve style compatibility, which can be used in daily project development.
  • Some styles refer to normalize.css
  • There are good suggestions that can be fed back to me at Issues, and I will be happy to adopt them.
